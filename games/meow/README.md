# 喵喵喵 · B站播放器

一个极简、高颜值的 B 站视频单页播放器。  
支持输入任意 BV 号实时切换，标题文字自带“呼吸灯”特效，夜间观感极佳。

---

## ✨ 特性

- 🎬 **即输即播**：输入 BV 号后点击按钮或回车即可加载视频
- 💡 **呼吸灯特效**：主标题“喵喵喵”以柔和光晕缓慢闪烁
- 🌓 **暗黑 UI**：渐变背景 + 卡片式布局，护眼又高级
- 📱 **响应式**：手机、平板、桌面多端自适应
- 🪶 **零依赖**：纯 HTML + CSS + 原生 JS，无需任何框架

---

## 🚀 快速开始

1. 克隆或下载本仓库
2. 双击 `index.html` 即可本地运行  
   （如部署到服务器，可直接托管整个目录）
   （不过这种小项目不值得部署到服务器吧）

---

## 📋 使用方法

| 操作 | 说明 |
| ---- | ---- |
| 输入 BV 号 | 支持格式：`BV1GJ411x7h7` |
| 播放 | 点击“播放”按钮或按键盘 `Enter` |
| 重置 | 清空输入框后再次点击播放，恢复默认视频 |

---

## 🛠️ 自定义

| 位置 | 说明 |
| ---- | ---- |
| `defaultBV` | 默认视频，修改 JS 顶部变量即可 |
| 主题色 | 调整 `:root` 中的 `--accent` 与 `--accent-hover` |
| 呼吸灯速度 | 修改 `@keyframes breathe` 的 `2.5s` 数值 |
