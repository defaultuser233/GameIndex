# 🩸 死亡测试（DeathTest）

一个氛围感十足的暗黑风格心理小测试，通过 10 道惊悚向选择题，评估你对「生命重视度」的隐藏倾向，并给出对应的「死亡人格」称号与解读。

## ✨ 特色

- 纯前端实现，零依赖，打开即用
- 暗黑哥特 UI：动态红黑色系、脉冲光效、边框呼吸灯
- 渐进式进度条 + 问题卡片淡入动画，沉浸感拉满
- 4 档结果人格（死神使者 / 灵魂行者 / 命运挑战者 / 生命守护者）
- 源码仅单 HTML，可离线运行，也可直接丢进任意静态托管

## 🚀 使用方式

- 直接双击 `deathtest.html` 即可开始测试。
- 或把文件丢到 GitHub Pages / Vercel / Netlify 等静态托管，访问链接即可。
- 无需后端、数据库或额外配置。

## 🧪 计分逻辑

- 每题 4 个选项，对应 1–4 分（越高越「惜命」）
- 满分 40 分，按 10-20 / 21-25 / 26-30 / 31-40 四段映射结果
- 若分数越界，自动取最接近区间，保证必有结果

## 🎨 自定义指南

| 要改的内容 | 快速定位 |
|------------|-----------|
| 题目/选项/分值 | 修改 `questions` 数组 |
| 结果文案与区间 | 修改 `results` 数组 |
| 配图 | 把 `result.image` 换成自己的图 |
| 主色调 | 搜索 `#ff0000` / `#8b0000` 批量替换 |
| 动效速度 | 调整 `@keyframes` 里的 `duration` |
